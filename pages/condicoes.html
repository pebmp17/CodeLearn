Condicionais

As estruturas condicionais são aquilo que permite ao programador ter a possibilidade fazer o seu programa ter certo comportamento apenas quando certas condições forem cumpridas. Um cliente só pode sacar dinheiro caso ele tenha algum na sua conta, por exemplo.

if

O if é a estrutura condição mais simples, porém também pode ser considerada a mais importante. Ele funciona na seguinte maneira: caso a expressão que está dentro do if seja verdadeira(true) ele executa o bloco de código dentro dele, caso contrário ele "pula" essa parte e começa a executar a próxima linha existente depois desse bloco.

else

O else só pode existir quando está dentro do mesmo bloco de código de um if, proxímo a ele e sem que haja outro else para esse mesmo if. O else determina o que irá ocorrer caso a expressão contida dentro do if não seja verdadeira(por isso necessita de um if para que ele possa funcionar). Agora você pode estar se perguntando: "Mas se quando a expressão do if não é verdadeira ele já não vai automaticamente para a proxíma linha de código depois do if sem ter um else, então para que que ele existe? A resposta é bem simples: caso a expressão do if seja verdadeira e não exista um else o programa irá executar tanto o existe dentro do if quando o que existe depois, o que não ocorreria com um else pois ele executaria ou o bloco de código dentro do if ou o bloco de código dentro do else, e não os dois.
